Materi Pemrograman

Pemrograman WEB menggunakan platform Edx sebagai media belajar, didalamnya berisikan 11 session yang mebahas berbagai pengetahuan dasar dalam membangun halam web. Materi yang dipelajari diantaranya adalah menengai Git, HTML, CSS, SCSS, Python, Flask, dan JavaScript.

Project Pemrograman

Terdapat beberapa project yang dapat dilakukan pada pemrograman WEB, diantaranya:

  1. Project 0, Project ini melatih penggunaan HTML dan CSS untuk mendesign halaman web dan untuk belajar menggunkan SCSS untuk menulis stylesheets yang lebih kompleks.
  2. Project 1, Tujuan Project ini untuk melatih penggunan python dan memperoleh pengalaman menggunakan flask serta belajar penggunaan SQL untuk berinteraksi dengan database.
  3. Project 2, Pada project ini diperlukan untuk membangun tampilan halaman web serta belajar menggunakan JavaScript yang berjalan di server-side.
